# Divine Health Website

A concise, GitHub Pages compatible static website for Divine Health.

## Files

- `index.html` - the website content
- `styles.css` - the design and responsive layout
- `assets/kamila-portrait.jpg` - edited portrait used in the hero
- `assets/gold-atmosphere.jpg` - soft gold atmosphere image used as a visual band
- `assets/hero-visual.svg` - fallback abstract visual asset

## Publish on GitHub Pages

1. Upload these files to a GitHub repository.
2. In the repository, go to **Settings > Pages**.
3. Set the source to the main branch and root folder.
4. Save. GitHub will provide the live website URL.
